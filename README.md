# ♟️ JavaScript Chess Game

A fully interactive chess game built with HTML, CSS, and JavaScript — featuring neon animations, hover effects, and a clean board UI.

## 🎮 Features
- Full 8x8 chess board with alternating colors
- Neon glow animations on pieces
- Smooth hover transitions
- Turn indicator (White / Black)
- Piece shake animations on interaction
- Built with vanilla JS + jQuery

## 📁 Folder Structure
chess/
├── index.html      ← Main game page
├── style.css       ← Board styling & animations
└── script.js       ← Game logic

## 🚀 How to Run
1. Clone the repository:
```bash
git clone https://github.com/Shreyaaaaaak/chess-game.git
```
2. Open `index.html` in your browser — no server needed!

## 🛠️ Built With
- HTML5
- CSS3 (neon animations, transitions)
- JavaScript (vanilla + jQuery 3.2.1)

## ⚙️ How to Play
- Click a piece to select it
- Valid moves will be highlighted in green
- Players alternate turns — White goes first
- The turn indicator updates after each move
